Input Clafer Model and Options

From this window, you can upload your .cfr file, select a predefined example, or enter the model directly using the source code editor.
  • The model must be expressed using Clafer.
  • Use the file selector to choose a file. Make sure the extension of the file you are trying to submit is .cfr. Use the button "" next to the file selector to upload and optimize the selected file.
  • Use the "" drop down to select an example and use the button "" next to the example selector to optimize the example. Check the checkbox "load in editor" to have the example loaded into the source code editor. Otherwise, the example will be compiled and instantiated without affecting the contents of the editor. This is useful for using examples as a reference.
  • Use the button "" directly above the source code editor to optimize the model from the editor.
  • Use the "Optimization backend: " drop down to select either the multi-objective, Alloy-based optimization backend (ClaferMoo) or the single-objective Choco-based one.
