In this tool, you can visualize and explore the set of non-dominated optimal variants (called Pareto front) computed by Clafer Multi Objective Optimizer (ClaferMoo). Press the '?' button of each window to see specific help. In general:
  • Use the view Input Clafer Model and Options to upload, select an example, or enter a Clafer model to optimize.
  • Use the view Matrix view to filter the instances by selecting and eliminating features.
  • Use the view Graph to visually explore the Pareto front.
  • Use the view VariantComparer to perform tradeoff analysis of a set of selected variants.
  • Use the view Control to filter by quality ranges.
  • Use the view Output to see messages.
